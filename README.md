# E-Commerce Management Platform (MERN Stack)

## Overview

A full-stack E-Commerce Management Platform built using the MERN Stack. The application provides secure authentication, product management, shopping cart functionality, order processing, and inventory management.

## Features

* JWT-based User Authentication and Authorization
* Role-Based Access Control (Admin/User)
* Product Management (Create, Read, Update, Delete)
* Shopping Cart Management
* Order Processing System
* Automatic Stock Updates After Successful Orders
* Order Status Tracking
* MongoDB Database Integration
* RESTful API Architecture

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT Authentication
* Mongoose ODM
* Postman (API Testing)
